# kal-public

Generated public dashboard for the private **kal-private** project-tracking hub
(part of the `kaleidoscope` workspace).

⚠️ **Do not edit by hand.** Everything here is produced by `kal-private/tools/build-dashboard.sh`,
which redacts the private tracking data down to a whitelist of safe fields before writing
`projects.json`. Manual edits will be overwritten on the next build.

Served via GitHub Pages.
